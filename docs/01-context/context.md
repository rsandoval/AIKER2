# Context

## Contexto Educacional

El aprendizaje moderno requiere:
- Evaluación remota
- Feedback rápido
- Escalabilidad

---

## Problema estructural

Las preguntas abiertas:
- Son más valiosas
- Pero difíciles de evaluar

---

## Limitaciones actuales

- Evaluación manual no escala
- Sistemas actuales usan preguntas cerradas
- Falta feedback útil

---

## Contexto tecnológico

- Avances en NLU permiten evaluar lenguaje natural
- Disponibilidad de modelos compactos (TLM)
- Infraestructura cloud accesible

---

## Restricciones

- Base instalada existente
- Limitaciones de performance actuales
- Equipo de desarrollo acotado

---

## Fuerzas del sistema

- Necesidad de automatización
- Escalabilidad
- Calidad del feedback
- Multicanalidad

---

## Oportunidad

Crear una plataforma capaz de evaluar conocimiento real,
no solo respuestas predeterminadas.
