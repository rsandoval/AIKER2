
# Project Overview

## Propósito

AIKER es un sistema que permite evaluar conocimiento humano a través de respuestas abiertas,
automatizando la corrección y generando feedback de alta calidad.

---

## Qué hace el sistema

- Genera pruebas a partir de preguntas definidas
- Recibe respuestas abiertas
- Evalúa semánticamente contenido
- Entrega feedback detallado

---

## Problema que resuelve

Evaluar respuestas abiertas es:
- Costoso
- Lento
- Inconsistente

AIKER resuelve esto mediante evaluación automática basada en lenguaje natural.

---

## Diferenciador Clave

No evalúa palabras clave, sino:
- Ideas
- Argumentos
- Estructura conceptual

---

## Evolución

El proyecto incorpora:

- Escalabilidad
- Multicanalidad
- Mejoras en modelos de evaluación

---

## Criterios de éxito

- Evaluación consistente
- Feedback útil
- Soporte concurrente alto
- Uso en múltiples canales
